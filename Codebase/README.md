### Codebase
This folder contains entire Codebase for COSE.

#### Dependencies:
* Node.js
* MySQL Server
* Python (Version>2.7)
* Python Packages:
  * dateutil
  * numpy
  * scipy
  * pandas
  * httplib2
  * MySQL-python
  * sqlalchemy
  * simplejson
  * config
  * requests
  * google-api-python-client
  * scikit-learn
  * ntlk (After installing this package, run command "sudo python -m ntlk.downloader -d /usr/share/ntlk_data all" to install ntlk-data)
  * feedparser
  * beautifulsoup4
  * lxml
  * Flask
  * Flask-Login
  * PyOpenSSL
